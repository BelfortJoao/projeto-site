# Projeto Site / Site Project

Este repositório contém os arquivos utilizados para criar um site básico de um curso de HTML disponível no YouTube.

This repository contains the files used to create a basic website for an HTML course available on YouTube.

## Descrição / Description

O projeto consiste em um site simples desenvolvido com HTML e CSS para o curso básico de HTML.

The project consists of a simple website developed with HTML and CSS for the basic HTML course.

## Conteúdo / Content

- `index.html`: Página inicial do site.
- `style.css`: Folha de estilos CSS para o site.
